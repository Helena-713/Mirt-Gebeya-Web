# Mirt-Gebeya-Web

## Project Description
Mirt Gebeya is a responsive e-commerce platform connecting Ethiopian farmers directly with consumers. This full-featured marketplace allows users to browse fresh produce, manage a shopping cart, and complete secure checkoutâ€”all while supporting local agriculture with fair pricing.

##  Features
- **Multi-page Website**: Home, About, Products, and Cart pages
- **Product Catalog**: Browse vegetables, fruits, and herbs with search & filter
- **Shopping Cart**: Add/remove items, update quantities, view totals
- **User Authentication**: Login and registration forms with validation
- **Responsive Design**: Mobile-first approach works on all devices
- **Local Storage**: Cart and user data persist across sessions
- **Accessibility**: WCAG compliant with semantic HTML and ARIA labels

## Technologies Used
- **HTML**: Semantic markup with accessibility features
- **CSS**: Custom CSS with Flexbox, CSS Grid, and CSS Variables
- **JavaScript**: Vanilla JS with DOM manipulation and localStorage
- **Font Awesome**: Icons for enhanced UI
- **Google Fonts**: Typography styling

## How to Run the Project

1. Download all project files
2. Open `index.html` in any modern web browser
3. Navigate between pages using the header links
